# Speaker Identity Data

This directory stores committed, non-audio speaker identity metadata.

The Chris Bache reference manifest records public source/time ranges from
already timecoded diarist files where the speaker label identifies Chris Bache.
It does not contain voice embeddings or audio clips.

Generated audio clips and speaker embeddings are biometric-like working
artifacts. Keep them local under ignored build/report paths unless the archive
owner explicitly changes that policy.
